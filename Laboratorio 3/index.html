  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    • El internet es la red de computadores a la cual se conectan nuestros dispositivos y la world wide web es la coleccion de paginas que hay en ella.

  • ¿Cuáles son las partes de una URL?

    • Las partes son el protocolo + subdominio + dominio + ruta + parametros

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • GET: Se usa para conseguir información de una pagina o servidor. POST: Se usa para mandar datos a un servidor HEAD: Funciona como GET pero no obtiene el body de la pagina PUT: Modifica el recurso completo PATCH: Solamente modifica parte de un recurso DELETE: Elimina el recurso completo

Laboratorio 3

  • Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    • !important sirve para dar prioridad a un elemento. Por ejemplo si una clase tiene un color de fondo pero despues a otra clase le pones !important todos los colores de fondo van a ser del que marcaste que era importante. Esta funcion sirve en casos en que tienes que cambiar un estilo que no se puede cambiar de ninguna otra manera.

  • Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    • Porque una a veces los fondos no se comportan bien con todas las resoluciones de pantalla haciendo que la pagina aparesca de manera incorrecta.

  • Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    • Hay medidad que son absolutas y otras relativas. Las absolutas es mejor utilizarlas cuando sabemos exactamente el ambiente en el que sera utilizado nuestra pagina y las relativas dependen de otros valores por lo que es mejor utilizarlas en una pagina web que se utilizara en difernetes dispositivos.
